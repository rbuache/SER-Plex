Décomposition des dossiers:


Workspace : il s'agit du workspace eclipse directement ouvrable.

	Office : contient l'application de l'office du cinéma.
		Cette application contient aussi les xml et dtd officiel.
		XML
		DTD
	Admin  : contient l'application de l'administrateur.
	Media  : contient l'application des média.

XML: contient la base de donnée de l'administrateur cinéma

DTD: contient les dtd créées à la première étape.
